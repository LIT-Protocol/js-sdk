# Supported EVM Chains

## Intro

Lit Protocol supports access control conditions across **78 EVM-compatible blockchains**. This means you can create conditions that check balances, token ownership, NFT holdings, and smart contract states across multiple networks.

Each supported chain uses its native cryptocurrency for balance checks and can be referenced by its unique **Lit Identifier** when building Access Control Conditions.
